# Shivam Gupta Portfolio

This is a personal portfolio website built with React and Vite.

## Features
- Responsive design
- Pages: Home, About, Projects, Contact
- Client-side routing (React Router)

## Getting Started

### Install dependencies
```sh
npm install
```

### Start development server
```sh
npm run dev
```

### Build for production
```sh
npm run build
```

### Preview production build
```sh
npm run preview
```

## Folder Structure
```
portfolio/
  src/
    pages/
      Home.jsx
      About.jsx
      Projects.jsx
      Contact.jsx
    App.jsx
    main.jsx
    index.css
    App.css
  package.json
  vite.config.js
  README.md
```

## Deployment
You can deploy the `dist` folder to any static hosting service (e.g., GitHub Pages, Vercel, Netlify).
